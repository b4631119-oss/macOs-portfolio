# 💻 Interactive macOS Portfolio

Интерактивное веб-портфолио, выполненное в стиле операционной системы macOS. Проект объединяет мои ключевые фронтенд-работы в единую экосистему с симуляцией рабочего стола, оконного менеджмента и встроенного терминала.

## 🚀 Стек технологий

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **UI Components:** Shadcn UI & Lucide React
* **Deployment:** Vercel

## 🛠️ Особенности реализации и рефакторинг

Проект был детально переработан и оптимизирован для достижения максимальной стабильности:
* **Утечки памяти (Memory Leaks):** Внедрен `AbortController` для отмены незавершенных асинхронных `fetch`-запросов к API GitHub при быстром закрытии окон.
* **Контроль стейта:** Исправлен блок `finally` в хуках загрузки данных, предотвращая обновление состояния на размонтированных компонентах.
* **Типизация:** Полностью удалены небезопасные типы `any`. Архитектура данных строго типизирована через TypeScript интерфейсы (например, `GitHubUser`).
* **UI/UX:** Исправлен конфликт двойных скроллов (`overflow`) внутри модальных окон, добавлен плавный Skeleton Loader для индикации загрузки профиля.

## 📁 Мои проекты внутри симулятора

Внутри портфолио интегрированы и доступны для просмотра мои реальные разработки:
1. **Cuaderno** — Цифровая тетрадь для заметок разработчиков.
2. **Real-Time Chat App** — Чат-приложение на Next.js с базой данных Firebase Firestore в реальном времени.
3. **StarStream (Movie App)** — Сервис поиска фильмов с рефакторингом стейта на Redux Toolkit.
